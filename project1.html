Welland Minor Baseball League

This project was developed in collaboration with a team for the Welland Minor Baseball League to streamline league management and improve user experience for coaches, parents, and players. The project helped me to get a deeper understanding of data structures, front-end and back-end development using different programming languages to build and application applications that solves the needs of the client. My primary role in this project was contributing to the design and development of the web application, ensuring it was both functional and easy to navigate.

Key Features:


  • Player Registration: Developed forms to allow parents and coaches to register players, track their progress, and manage personal details.

  • Team Management: Built a feature to assign players to teams based on age and skill level, with the ability to view and manage team rosters.

  • League Scheduling: Implemented a scheduling system for games, including notifications for upcoming matches and results.

  • Database Integration: Utilized MySQL to store player and team information, and integrated backend logic to handle registration, scheduling, and player stats efficiently.

  • Responsive Design: The website was designed to be fully responsive, ensuring it works seamlessly on desktop and mobile devices.

  • Technologies Used: HTML/CSS, JavaScript (React.js), MySQL, Node.js, Express.js
